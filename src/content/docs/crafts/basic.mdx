## Create your very own button using the button press!
1. Ask a PI at front desk to grab a set of button parts. You can choose what size you want!
2. Create a design using Illustrator, Photoshop, Canva, Paint, etc. Make sure the size of the design is matches the button you have. HINT: Drawing a circle outline (the same size as your button) makes it much easier to cut out later.
3. Print out your design and cut it out using scissors or the vinyl cutters (force= 10).
4. Follow the instructions at the button maker station in the Crafts area to assemble your button!
5. Show your new button to a front desk PI to get a stamp!
